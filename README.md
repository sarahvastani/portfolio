## AlphaPulldown: AI-Driven Modeling of Microbial Protein–Protein Interactions

**Tools:** AlphaFold-Multimer, AlphaPulldown, Python, Linux/HPC, SLURM, structural bioinformatics

Developed a high-throughput computational workflow using AlphaPulldown, an AlphaFold-Multimer-based framework, to predict protein–protein interactions in *Escherichia coli*. The project focused on generating and evaluating structural models to identify biologically plausible interaction candidates for experimental validation.

Key contributions:
- Generated large-scale protein complex predictions using GPU-enabled HPC clusters.
- Managed computational workflows using Linux environments and SLURM workload scheduling.
- Evaluated predicted interactions using structural confidence metrics, membrane localization, protein topology, and functional annotations.
- Organized and analyzed large structural datasets including PDB, PKL, and JSON model outputs.
- Integrated computational predictions with experimental biological knowledge to prioritize candidate interactions.

## Predicting Migration Survival in Swainson’s Thrushes Using Machine Learning

**Tools:** Python, scikit-learn, Random Forest, Balanced Random Forest, Neural Networks, SHAP, KNN Imputation, SMOTE

Developed an end-to-end machine learning pipeline to predict migratory survival in Swainson’s thrush hybrids using integrated genetic, morphological, and behavioral datasets.

Key contributions:
- Built predictive models using Balanced Random Forest and neural network approaches.
- Processed biological datasets containing over 500 samples through feature engineering, missing data imputation, and class imbalance correction.
- Applied KNN imputation and SMOTE while implementing data leakage prevention strategies.
- Interpreted model predictions using SHAP values, permutation importance, and feature importance analyses.
- Designed prevalence-matched null simulations to evaluate model performance and biological significance.

Published in *Scientific Reports*:
"Predicting Migratory Survival in a Songbird Hybrid Zone Using Machine Learning" (2026)

## Circadian Rhythms in *Lumbriculus variegatus*

**Techniques:** Immunohistochemistry, fluorescence microscopy, microscopy image analysis, histology

Investigated circadian protein expression patterns in the annelid worm *Lumbriculus variegatus* using immunohistochemistry and fluorescence microscopy.

Key contributions:
- Performed TIM and 5-HT antibody staining across six time points in a 24-hour light-dark cycle.
- Prepared and analyzed tissue samples to evaluate protein localization.
- Used fluorescence microscopy to characterize expression patterns within neural tissues.
- Managed experimental workflows involving more than 50 biological specimens.
